# 🎛️ Admin Dashboard Quick Start Guide

## Accessing the Admin Dashboard

### Navigate to Admin Panel
1. Log in with your admin account
2. Go to `/admin` or click admin link in navigation
3. You'll see the new **Dashboard** tab

## Main Dashboard Overview

When you first enter, you'll see four key stats:
- **📅 Active Events** - Total events you've created
- **📝 Blog Posts** - Total blog posts
- **👥 Total Members** - Registered users
- **📬 Guest Requests** - Unregistered interest submissions

## Quick Start Tasks

### Creating an Event (5 minutes)

1. Click **"Create Event"** button in Quick Actions
2. A form will appear at the top of the Events section
3. Fill in the fields:
   - **Slug**: `my-awesome-event` (lowercase, no spaces)
   - **Title**: `My Awesome Event`
   - **Subtitle**: Short description
   - **Short Description**: 1-2 sentences
   - **Date & Location Labels**: When and where
   - **Banner Image URL**: Link to event image
   - **Host Name**: Company/organization name

4. For list items (About, Expectations, etc.):
   - Enter one item per line
   - System will automatically split them

5. For FAQs:
   - Format: `Question here || Answer here`
   - One FAQ per line

6. Click **"Create Event"**
7. See your event in the list below!

### Creating a Blog Post (5 minutes)

1. Go to **"Blogs"** tab
2. Click **"New Post"** button
3. Fill in:
   - **Slug**: `my-blog-post`
   - **Title**: Blog post title
   - **Excerpt**: 1-2 sentence summary
   - **Author**: Your name
   - **Date**: Publication date
   - **Read Time**: Estimated reading time (e.g., "5 min")
   - **Cover Image URL**: Blog header image

4. For sections:
   - Format: `Heading || Content here`
   - One section per line
   
5. Tags:
   - One tag per line

6. Click **"Create Post"**

### Editing Content

1. Find the item in the list
2. Click the **"Edit"** button
3. Update the form fields
4. Click **"Update"** button

### Deleting Content

1. Find the item in list
2. Click **"Delete"** button
3. Confirm deletion
4. Item is removed

### Viewing Members

1. Go to **"Members"** tab
2. See all registered users with:
   - Name and email
   - User role
   - Location
   - Join date and last login

3. View quick stats on the right side

### Managing Guest Requests

In the **"Members"** tab, scroll down to see:
- Non-member event interest submissions
- Their contact info
- Which event they're interested in
- Their submission date

## Pro Tips 🚀

### Event Management
- Always use clear, SEO-friendly **slugs**
- Add at least one image
- Include all FAQ items for common questions
- Use consistent date/location labels

### Blog Writing
- Keep excerpts punchy and interesting
- Accurate read time helps readers
- Organize content in logical sections
- Use relevant tags for discoverability

### Quick Navigation
- Use **Dashboard** tab for overview
- Jump directly to **Events**, **Blogs**, or **Members**
- Click **"View Events"** or **"View Blog"** to see public pages
- Use browser back button to return

### Data Entry Tips
- Use line breaks to separate list items
- Format FAQs carefully: `question || answer`
- Save frequently for large forms
- Confirm you have image URLs ready

## Common Issues & Solutions

### Form Won't Submit?
- Check that all required fields are filled
- Ensure **Slug** is unique (not already used)
- Verify URLs are valid (http:// or https://)

### Can't Find Member?
- Go to Members tab to see all users
- Look for their name or email

### Need to Delete?
- Click Delete button
- Confirm when prompted
- Deletion is permanent!

### Unsure About Format?
- FAQs: `Question || Answer`
- Sections: `Heading || Content`
- Lists: One item per line
